# Blog Example for Codeception Learning

## Create project

```bash
laravel new blog
cd blog
```

[edit] `composer.json`

```json
    "require": {
        "laravel/framework": "5.0.*",
        "illuminate/html": "*"
    },
    "require-dev": {
        "phpunit/phpunit": "~4.0",
        "phpspec/phpspec": "~2.1",
        "codeception/codeception": "~2.0.12"
    },
```

2.0.11+ Support Laravel5

```
composer update
```
